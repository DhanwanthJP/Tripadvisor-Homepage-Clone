# Tripadvisor-Homepage-Clone

A responsive front-end clone of the Tripadvisor homepage built using HTML and CSS.
This project replicates the layout, structure, and styling of the official Tripadvisor landing page for learning and practice purposes.

## Project Overview

### This project focuses on recreating the visual structure of the Tripadvisor homepage including:

Navigation bar\
Caption tagline\
Categories section (Search All, Hotels, Things to Do, Restaurants)\
Search bar with responsive behavior\
Hero section layout\
Footer section\
Centered container layout with max-width design\
Styled buttons and hover effects\
Animation for caption\

### The goal of this project was to practice:

Flexbox layout\
Responsive design techniques\
CSS positioning\
Shorthand properties\
Modern UI structure\
Clean and scalable CSS organization\
Features\
Responsive centered layout using max-width\
Flexbox-based navigation and search bar\
Custom styled buttons\
Icon integration using Font Awesome\
Proper spacing and alignment\
Hover and focus styling\
Clean UI inspired by the original website

### Technologies Used

**HTML5
CSS3**

**The layout adapts to different screen sizes by:**

Using flex: 1 for flexible elements\
Avoiding fixed widths where possible\
Applying max-width for container control\
Maintaining proper spacing with padding instead of fixed heights\
